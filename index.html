Merge Sort Algorithm

What is Merge Sort?

Merge Sort is a Divide and Conquer algorithm that splits an array into smaller halves, sorts them individually, and then merges them back together. It is efficient and stable with a time complexity of O(n log n).

    if size of arr <= 1:
        return arr

    mid = size of arr / 2
    left = MergeSort(arr[0:mid])
    right = MergeSort(arr[mid:end])

    return Merge(left, right)

Merge(left, right):
    result = []
    while both left and right have elements:
        if left[0] <= right[0]:
            append left[0] to result
            remove left[0]
        else:
            append right[0] to result
            remove right[0]

    append remaining elements of left
    append remaining elements of right

    return result

Time Complexity

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n log n)
